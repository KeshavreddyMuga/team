# Team Workspace Organizer (Flask + SQL + Mail + SocketIO)

This is a Flask-based online team workspace organizer where team members can:
- Create projects
- Upload weekly files
- Move through project weeks together
- Receive email notifications
- Use real-time updates via Socket.IO
